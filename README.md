# MckSynth

## Description

This is just a simple app for learning purposes.
Used libs:
- RtMidi
- RtAudio
- Gtk+3

## Dev Dependencies

```
sudo apt install librtaudio-dev librtmidi-dev 
sudo apt install libgtk-3-dev libgtkmm-3.0-dev
```
